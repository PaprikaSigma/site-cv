DESCRIERE
---------
Proiect web realizat în cadrul stagiului de practică la IP Centrul de Excelență
în Economie și Finanțe (CEEF), specialitatea Dezvoltarea Aplicațiilor Web.
Site-ul include un CV personal interactiv, un portofoliu de sarcini practice,
pagini educaționale și documentele aferente stagiului de practică.


STRUCTURA PROIECTULUI
---------------------
index.html              — CV personal (pagina principală)
sarcini.html            — Sarcini Practică (dropdown cu cod, imagini, linkuri)
relatii-poo.html        — Relațiile dintre clase în POO (teorie + exemple C#)
planificare.html        — Planificarea activităților zilnice (Cap. 1)
experiente.html         — Experiențe acumulate (Cap. 2)
declaratie.html         — Declarația privind asumarea răspunderii (Cap. 3)

Imagini/
  poza.jpg              — Fotografia de profil
  CV-bg.png             — Imaginea de fundal a header-ului

CV Dragoș Pelin.pdf     — CV-ul în format PDF (descărcabil)


TEHNOLOGII FOLOSITE
-------------------
- HTML5
- CSS3 (Flexbox, Grid, animații, variabile CSS)
- Font Awesome 6.5.1 (iconițe)
- Google Fonts — Poppins, Lora, Fira Code
- Fără framework-uri sau biblioteci JavaScript externe


PAGINI — DESCRIERE SCURTĂ
--------------------------
index.html
  Pagina principală cu informații despre mine, educație, competențe tehnice
  (bare de progres), realizări cu linkuri către proiecte, limbi străine,
  hobby-uri și calități. Include buton de descărcare CV.

sarcini.html
  Conține toate sarcinile de practică (M1.5, M2.9, M3.18, M3.28, M4.2, M5.2,
  M6.3, POO_8, POO_21, Independent, Sarcina Suplimentară) în secțiuni dropdown.
  Fiecare secțiune permite adăugarea unui link OnlineGDB, încărcarea unei imagini
  cu rezultatul din consolă și inserarea codului sursă.

relatii-poo.html
  Pagină educațională despre relațiile dintre clase în POO:
  Moștenire, Agregare, Compunere și Dependență — cu diagrame UML simple,
  exemple de cod C# cu sintaxă colorată și tabel comparativ.

planificare.html
  Jurnal zilnic al activităților de practică (Zilele 1–10). Fiecare zi afișează
  obiectivele planificate, sarcinile realizate și comentariile aferente,
  structurate pe trei coloane. Zilele fără sarcini sunt marcate distinct.

experiente.html
  Rezumat al experiențelor și competențelor dobândite pe parcursul stagiului:
  algoritmi C++, POO în C#, dezvoltare web HTML/CSS, documentare tehnică,
  aptitudini transversale și concluziile personale ale elevului.

declaratie.html
  Document oficial de asumare a răspunderii pentru materialele prezentate
  în raportul de practică. Conține câmpuri pentru dată, semnătură și nume.


CUM SE RULEAZĂ
--------------
1. Descarcă sau clonează proiectul într-un folder local.
2. Asigură-te că folderul "Imagini/" conține fișierele poza.jpg și CV-bg.png.
3. Deschide index.html într-un browser modern (Chrome, Firefox, Edge).
4. Navighează între pagini folosind meniul de navigare din header.

Nu este necesar niciun server local — proiectul rulează direct din browser.
